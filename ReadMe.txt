Este script Python simplifica a rotina de atualizar uma tabela dinâmica específica em uma planilha do Excel.

FUNCIONALIDADES
- Localiza a tabela: Identifica a tabela dinâmica com o nome especificado na planilha indicada.
- Atualiza os dados: Executa a atualização da tabela dinâmica, garantindo que os dados estejam sempre atualizados.

COMO USAR
- Instale a biblioteca: Execute pip install openpyxl no terminal para instalar a biblioteca necessária.
- Configure o caminho: Substitua o caminho da planilha no código pela localização exata do seu arquivo.
- Execute o script: Rode o script Python para iniciar a automatização.
